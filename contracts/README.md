# Leo Smart Contract

## Deployment

1. Build the contract:
```bash
leo build
```

2. Deploy to testnet via Leo Playground:
- Go to https://play.leo-lang.org/
- Paste the contract code
- Deploy to testnet

## Program ID

`onyx_v1.aleo`

## Functions

- `mint_artifact` - Brand mints a new product passport
- `transfer_artifact` - Owner transfers to new owner
- `report_stolen` - Owner marks artifact as stolen
- `prove_for_resale` - Generate resale proof token
